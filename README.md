# Zexiang Zhang Portfolio

A personal portfolio website built with **React**, **Vite**, and **Three.js**, showcasing projects, experience, and interactive 3D UI elements.

---

## 🚀 Tech Stack

- React 18
- Vite
- Tailwind CSS
- Three.js / React Three Fiber
- Framer Motion
- React Router
- EmailJS

---

## 📦 Features

- Modern responsive UI
- 3D interactive components using Three.js
- Smooth animations with Framer Motion
- Project showcase section
- Contact form with EmailJS integration
- Mobile-friendly design

---

## 🛠️ Installation

Clone the repository:

```bash
git clone https://github.com/626-Legendary/zexiang-zhang-portfolio.git
cd zexiang-zhang-portfolio
````

Install dependencies:

```bash
npm install
```

---

## 💻 Development

Run local dev server:

```bash
npm run dev
```

App will be available at:

```
http://localhost:5173
```

---

## 📦 Build

Create production build:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

---

## 🌐 Deployment (Vercel)

This project is optimized for deployment on Vercel.

1. Push repo to GitHub
2. Import project into Vercel
3. Set build command:

   ```
   npm run build
   ```
4. Set output directory:

   ```
   dist
   ```
